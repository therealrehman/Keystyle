# KeyStyle Keyboard 🎹

Samsung Keys Cafe inspired production-level Android keyboard with:
- 🎨 4-Color Radial Burst Animation (Hero Feature)
- 🖌️ Live Theme Studio
- ⌨️ Keyboard Layout Builder
- 😊 Offline Emoji + Clipboard + Voice
- ⚡ GPU-Accelerated Animations
- 🔒 Privacy-First (No Data Collection)

## Tech Stack
- Kotlin + Jetpack Compose
- MVVM + Hilt DI
- Material 3
- DataStore Preferences

## Features
- **Phase 1:** Core IME Engine + QWERTY/Symbols/Numbers
- **Phase 2:** Theme Studio + Live Preview
- **Phase 3:** 4-Color Radial Burst Animation
- **Phase 4:** Keyboard Builder + Drag/Resize
- **Phase 5:** Emoji + Clipboard + Voice modules
- **Phase 6:** Performance optimization + Play Store ready

## Build
```bash
./gradlew assembleDebug
```
